# providers-service

install node

npm install

install mongodb

To try service

1. node app.sj (http://localhost:3002/api/providers)
2. Use a Postman to send a POST with this JSON:
   {
  "provider": 1,
  "name": "Petshop",
  "address": "Calle 95 No 15 20",
  "city":  "Bogota",
  "neighborhood":  "Chico"
}
